# Code of Conduct

## Our standard

We want this project to be welcoming, respectful, and helpful.

Examples of behavior that we expect:

- Be respectful and constructive
- Assume good intent
- Give feedback about the code, not the person
- Help newcomers feel comfortable asking questions

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Trolling or deliberately disruptive behavior
- Publishing private information without permission
- Repeatedly dismissing or mocking contributors

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other contributions that do not follow this code of conduct.

## Reporting

If someone is behaving in a way that harms the community, contact the maintainer through GitHub.

## Scope

This code of conduct applies to issues, pull requests, discussions, and other project spaces.
