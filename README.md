## Overview

Warframe Companion App is an Electron-based desktop utility focused on quality-of-life workflows for players:

- Track mastered and unmastered items by category
- Search and filter large item collections quickly
- View item details, drops, and resource requirements
- Access Warframe Market and trading tools from one place
- Monitor Prime Resurgence rotations and relic highlights
- Use MR Calculator and progress stats to plan mastery goals

## Features

- Modern category-driven item explorer
- Mastery progression ring and percentage completion metrics
- Real-time filtering (All, Mastered, Unmastered)
- Item detail modal with acquisition and crafting context
- Prime Resurgence panel with countdown and relic cards
- Trade Mode toggle and market integration
- Native Electron window controls and update checks
